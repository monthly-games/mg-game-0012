# MG-0012: Raid Event UI/UX 스펙

> **게임명**: Raid Event (레이드 이벤트)
> **장르**: 레이드 전투 RPG
> **카테고리**: Year 1 Core
> **문서 버전**: 1.0.0

---

## 1. 화면 설정

| 항목 | 값 |
|------|-----|
| **방향** | Portrait (세로) |
| **기준 해상도** | 1080 × 1920 |
| **지원 비율** | 16:9 ~ 21:9 |
| **타겟 FPS** | 60fps |

### 방향 선택 이유
- 팀 전투 UI 세로 배치
- 스킬 선택 바 하단 배치
- 보스 연출 상단 배치

---

## 2. Safe Area

| 영역 | 값 | 용도 |
|------|-----|------|
| 상단 | 44-62dp | 보스 HP, 타이머 |
| 하단 | 34-48dp | 스킬 바 |

## 3. HUD 레이아웃

```
┌─────────────────────────────────────┐
│  [BOSS: 드래곤]  [⏱ 2:45]    [⚙️]  │
│  [HP████████████████░░░░░░ 72%]    │  ← 보스 HP
├─────────────────────────────────────┤
│  ┌─────────────────────────────┐    │
│  │                              │    │
│  │         🐉 BOSS             │    │
│  │                              │    │
│  │    [WEAK: 불]               │    │  ← 전투 영역
│  │                              │    │
│  │  🗡️  🏹  🔮  🛡️             │    │
│  │  [아군 4인 파티]             │    │
│  └─────────────────────────────┘    │
├─────────────────────────────────────┤
│  [캐릭1] [캐릭2] [캐릭3] [캐릭4]    │  ← 캐릭터 상태
│  HP░░   HP██   HP██   HP██         │
├─────────────────────────────────────┤
│  [스킬1] [스킬2] [스킬3] [궁극기]   │  ← 스킬 바
│   CD:3    CD:0    CD:5    CD:0     │
└─────────────────────────────────────┘
```

### 3.1 보스 영역

| 요소 | 크기 | 설명 |
|------|------|------|
| 보스 HP 바 | 가로 전체 × 24dp | 체력 게이지 |
| 타이머 | 80×32dp | 제한 시간 |
| 약점 표시 | 64×24dp | 속성 약점 |

### 3.2 스킬 바

| 요소 | 크기 | 설명 |
|------|------|------|
| 스킬 버튼 | 72×72dp | 4개 스킬 |
| 쿨다운 표시 | 오버레이 | 남은 턴 |

---

## 4. 터치 영역

| 동작 | 영역 | 제스처 |
|------|------|--------|
| 스킬 사용 | 72×72dp | 탭 |
| 캐릭터 선택 | 캐릭터 영역 | 탭 |
| 보스 약점 확인 | 보스 영역 | 롱프레스 |
| 타겟 변경 | 적 영역 | 탭 |

### 터치 피드백
- 스킬 사용: 이펙트 + 데미지 숫자
- 궁극기: 컷인 연출
- 보스 패턴: 경고 표시
