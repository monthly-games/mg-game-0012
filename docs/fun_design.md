## 2. 코어 루프 (Core Loop)

```
┌─────────────────────────────────────────────────────────┐
│  1. 레이드 준비 (파티/장비 세팅)                         │
│         ↓                                               │
│  2. 레이드 전투 참여                                     │
│         ↓                                               │
│  3. 개인/전체 보상 획득                                  │
│         ↓                                               │
│  4. 이벤트 스토리/콘텐츠 소비 ─────────────→ (1로 반복)   │
└─────────────────────────────────────────────────────────┘
```

### 루프 상세 설명

1. **레이드 준비**: 파티 구성, 장비 세팅, 버프 아이템 준비
2. **레이드 전투**: 협동 레이드 보스 전투 참여
3. **보상 획득**: 개인 기여도 + 서버/길드 진행도 보상
4. **이벤트 소비**: 시즌 스토리, 특별 콘텐츠 경험

## 3. 재미 축 (Fun Pillars)

### 3.1 협동/공동 목표 (Co-op)
- **목표**: 함께 싸우는 일체감과 성취감
- **핵심 경험**: 우리가 함께 보스를 쓰러뜨렸다는 공동 성취
- **주요 요소**:
  - 역할 분담 (탱/딜/힐)
  - 길드/서버 진행도
  - 협동 스킬/시너지
  - 공동 보상

### 3.2 단기 몰입 (Short-term Engagement)
- **목표**: 한정된 기간 동안의 집중된 재미
- **핵심 경험**: 지금 아니면 못한다는 긴박감
- **주요 요소**:
  - 기간 한정 콘텐츠
  - 일일/주간 미션
  - 진행도 바
  - 카운트다운 타이머

### 3.3 한정 보상 (Limited Reward)
- **목표**: 이 시즌에만 얻을 수 있는 특별함
- **핵심 경험**: 한정판을 손에 넣었다는 만족감
- **주요 요소**:
  - 시즌 한정 캐릭터/스킨
  - 업적 칭호
  - 기념 아이템
  - 랭킹 보상

## 4. 디자인 레버 (Design Levers)

### 4.1 역할 분담 레이드
| 레버 | 설명 |
|------|------|
| 역할 기반 패턴 | 역할 분담(탱/딜/힐/서포트)을 전제로 레이드 패턴 설계 |
| 구현 방향 | 탱커 어그로 필수 패턴, 힐러 디버프 해제, 딜러 타이밍 |

### 4.2 이중 보상 구조
| 레버 | 설명 |
|------|------|
| 개인 + 공동 보상 | 개인 기여도 보상 + 서버/길드 누적 진행도 보상을 동시에 제공 |
| 구현 방향 | 개인 DPS 랭킹 보상 + 서버 전체 피해량 목표 달성 보상 |

### 4.3 테마 연출
| 레버 | 설명 |
|------|------|
| 단기 몰입 연출 | 연말/테마 연출과 스토리 컷신으로 단기 몰입 유도 |
| 구현 방향 | 연말 분위기 UI, 캐릭터 대화, 스토리 컷신, 특별 BGM |

### 4.4 크로스 프로모션
| 레버 | 설명 |
|------|------|
| 카메오 등장 | 다른 게임 캐릭터/요소 카메오 등장으로 크로스 프로모션 허브 역할 설계 |
| 구현 방향 | game_0001 영웅 게스트 참전, game_0004 카페 배경 스테이지 |

## 5. 재미 측정 지표 (KPI)

| 지표 | 목표 | 측정 방법 |
|------|------|-----------|
| 이벤트 참여율 | 80% 이상 | 레이드 1회 이상 참여 유저 비율 |
| 일일 레이드 횟수 | 5회 이상 | 레이드 참여 로그 |
| 길드 참여율 | 60% 이상 | 길드 레이드 참여 유저 비율 |
| 이벤트 완주율 | 50% 이상 | 메인 스토리 완료 비율 |
| 이벤트 기간 리텐션 | 70% 이상 | 이벤트 기간 중 잔존율 |
